КАК ДОБАВИТЬ БАННЕР
===================

1. Создай новую папку с номером: 04, 05, 06...
2. Положи в неё файлы:
   - image.jpg  (или .png, .webp) — картинка баннера (необязательно)
   - text.txt   — текст баннера (обязательно)

ФОРМАТ text.txt:
----------------
Первая строка → Заголовок баннера
Вторая строка → Текст описания
tag: Новинка
tag_color: #2e7d32
link: https://electronmir.ru/...
bg: #e8f5e9

ДОСТУПНЫЕ ЦВЕТА ТЕГОВ:
  Новинка  → #2e7d32  (зелёный)
  Акция    → #c62828  (красный)
  Новости  → #1565c0  (синий)
  Важно    → #e65100  (оранжевый)

ЦВЕТА ФОНА (bg):
  Зелёный  → #e8f5e9
  Оранжевый→ #fff3e0
  Синий    → #e3f2fd
  Серый    → #f5f5f5

После изменений: запусти build_html.py → git push
(или напиши менеджеру AI-ассистента)

ПОРЯДОК баннеров = порядок папок (01, 02, 03...)
Чтобы СКРЫТЬ баннер — переименуй папку, добавив _ в начало: _04
